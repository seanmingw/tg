# API文档

# 开发前必读

  1. 接口认证和鉴权

    除了开放接口，其余接口必须设置请求头Authorization用于认证和鉴权

    Authorization值格式为Bearer ${Token}

    ${Token}可通过登录接口（/api/auth/login）获得

  2. 通用返回对象

    返回示例:

    {
  "code": 20000,
  "type": "success",
  "message": "",
  "data": {}
}

    参数说明:

参数 必须 说明
code 是 状态码 20000/成功 50000/系统错误 50014/登录过期 50008/认证失败 50401/未认证 50403/暂无权限
type 是 类型 success/成功 error/错误 warning/警告
data 否 返回值
