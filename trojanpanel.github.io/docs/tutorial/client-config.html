# 客户端配置

# v2rayN

下载v2rayN-Core (opens new window)

# 配置Hysteria（Windows）

  1. 下载Hysteria (opens new window)

  2. 将Hysteria执行文件复制到v2rayN-Core目录下

  1. 编辑Hysteria配置文件
{
  "server": "域名:端口",
  "protocol": "udp",
  "up_mbps": 100,
  "down_mbps": 100,
  "auth_str": "密码",
  "socks5": {
    "listen": "127.0.0.1:1080"
  }
}

备注：auth_str不需要base64编码

  1. 开启对应的udp端口或者关闭防火墙

  2. 打开v2rayN，添加自定义服务器

  1. 将v2rayN的系统代理设置为自动配置系统代理

  2. 选中节点，点击Enter即可连接节点

# 配置NaiveProxy（Windows）

  1. 下载NaiveProxy (opens new window)

  2. 将NaiveProxy执行文件复制到v2rayN-Core目录下

  1. 编辑NaiveProxy配置文件
{
  "listen": "socks://127.0.0.1:1080",
  "proxy": "https://user:pass@example.com:port"
}

  1. 开启对应的tcp端口或者关闭防火墙

  2. 开v2rayN，添加自定义服务器

  1. 将v2rayN的系统代理设置为自动配置系统代理

  2. 选中节点，点击Enter即可连接节点
